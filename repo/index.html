welcome

Welcome to the official Wavelink Studios public Cydia repo. You will find all of the Cydia packages for Wavelink Studios here. You can even add the repo straight to Cydia/Zebra/etc with the button below. You dont even need to be jailbroken to see the latest packages and updates from us! Featured packages are listed on the home page below and all packages can be seen underneath them on their own page. Thank you for checking out our repo and packages.

Featured Package

Our currently featured package is BundleIDsXI (No App Icons). This is an app that allows you to see the Bundle ID's of all the applications installed on your device. Supports iOS 11-13.3
