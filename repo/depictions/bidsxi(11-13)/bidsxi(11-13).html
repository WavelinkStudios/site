Description

This app allows you to see the Bundle ID's of all the applications installed on your device.

This app was originally by @NoahSaso. Sadly it appears that he has since gone inactive around late 2017 and the app was only ever compiled for 32-bit devices. I figured this would be a relatively easy first attempt at jailbreak projects seeing as I have never really worked in this scene but have a little bit of coding knowledge from other types of small personal projects.

This project would not be where it is today without all of the amazing help from @Skittyblock, @TheTomMetzger, @EthanWhited, @MTAC8, and @xerusdesign.

Skitty and Tom both helped me get BundleIDsXI up and running for when iOS 11 was finally jailbroken.

MTAC8 helped me get BundleIDsXI up and running without app icons initially for iOS 13 to combat an issue with Ryan Petrich's AppList when iOS 13 was first jailbroken.

Ethan helped get the iOS 13 update working with app icons still using Ryan Petrich's AppList for iOS 13.

Xerus made the amazing new icon for BundleIDsXI when it first came out and the same one we still use today.

Staying true to the original project, I have made my update open source for all to take a look at and even help me learn by helping me update the app further. You can check out my fork of the app on my GitHub page. Major Updates from Noah's last build:

  • A10+ Device Support
  • Notched Device Support
  • Support for all iOS 11 to iOS 13 devices and firmwares
  • Removed all new internal applications that are never actually seen by the user
  • Changes to alert views
  • New "Copy All" feature
  • Brand new icon
  • Removal of deprecated code
