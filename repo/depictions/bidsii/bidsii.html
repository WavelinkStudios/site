Description

This app allows you to see the Bundle ID's of all the applications installed on your device.

This is a major update to BundleIDsXI which is also from us. We have left BundleIDsXI up as it works perfectly fine for all devices running iOS 11-12.4.4 and uses a different AppList. If you prefer to use Ryan Petrich's AppList and are on iOS 11-12.4.4 then please feel free to use BundleIDsXI instead. BundleIDsII and subsequently BundleIDsXI are both origianally based off of @NoahSaso's BundleIDs which no longer functions as of iOS 11.

An absolutely HUGE thanks to @Skittyblock and @TheTomMetzger for all of their help in getting this update working and tested. And finally a huge thanks to @xerusdesign for making the amazing new icon. Staying true to the original projects, I have made my update open source for all to take a look at and even help me learn by helping me update the app further. You can check out my source code over on github (coming soon).

Major Updates from BundleIDsXI:

  • Brand new name
  • Brand new app icon
  • Switched from Ryan Petrich's AppList to Spark's libSparkAppList
  • iOS 13+ Support
