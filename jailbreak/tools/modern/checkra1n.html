Home Tools Guides Wavelink Studios

Description

You can use checkra1n to jailbreak all A7-A11x devices running iOS 12.4-13.3.1 devices. This jailbreak is based on the checkm8 bootrom exploit. Technically this jailbreak cannot be patched at the base but can be broken by any major changes made to iOS itself. As of writing this iOS 13.4 breaks checkra1n but should be fixed in the future as development of the tool is still ongoing.
